# 我的第一篇博客文章

欢迎来到我的博客！这是使用自己搭建的极简静态博客生成器发布的第一篇文章。

## 关于这个项目

这个博客生成器遵循极简主义原则，只保留最核心的功能：

1. **Markdown 解析** - 支持标准 Markdown 语法
2. **静态页面生成** - 生成纯 HTML 页面，无需数据库
3. **简洁设计** - 专注于内容，而非复杂的样式

## 为什么选择静态博客

- **快速加载** - 纯 HTML 页面加载速度极快
- **安全性高** - 没有数据库，减少攻击面
- **易于部署** - 可以部署在任何静态托管服务上
- **版本控制友好** - Markdown 文件适合 Git 管理

## 未来计划

- 添加更多文章分类
- 优化移动端体验
- 考虑添加搜索功能

感谢阅读！
