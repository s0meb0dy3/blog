关于此博客

2025-09-27

欢迎来到我的博客！这是使用自己搭建的极简静态博客生成器发布的第一篇文章。

建站想法

创建这个博客是希望做一些属于自己的东西。看过hexo,hugo等框架，想来想去还是希望自定义的程度高一点，以后也能添加一些有趣的功能。我很喜欢一切可以自己修改，自己掌控的感觉，从0到1一点点添加功能，同时保持简洁与优雅，是我理想的博客实现。 同时我也希望做这个博客可以让我养成写作的习惯，记录生活或者学习的点点滴滴。

浩如烟海的互联网内容中竟然有我创建的一隅天地，想来还挺神奇的hh的😁

技术路线

选择的技术栈体现了简单高效的原则：

  • Node.js - 轻量级运行环境，生态完善
  • markdown-it - 成熟稳定的 Markdown 解析器
  • 原生 HTTP 服务器 - 无需额外框架依赖
  • 纯静态生成 - 最终输出为 HTML 文件
  • Git 版本控制 - 文章内容与代码统一管理
  • GitHub Pages - 免费稳定的托管服务

未来计划

基于当前架构，计划逐步完善功能：

  1. 功能扩展

    • 添加文章标签系统
    • 实现简单的搜索功能
    • 支持 RSS 订阅
    • 探索文章中加入图片格式

  2. 体验优化

    • 改进移动端适配
    • 添加深色模式支持
    • 优化页面加载速度

  3. 内容建设

    • 建立技术文章分类
    • 分享开发经验总结
    • 记录学习笔记

感谢您的关注，期待在这个简陋但实用的博客空间中与您分享更多内容！
